## 可选配置

| 参数               | 默认值                              | 说明                       |
| ------------------ | ----------------------------------- | -------------------------- |
| EXC_FOLDERS        | {.bzr,CVS,.git,.hg,.svn,.idea,.tox} | 设置 grep 命令要忽略的目录 |
| ZSH_CACHE_DIR      | $DZS/cache                          | 设置 zsh 的 cache 目录     |
| CASE_SENSITIVE     | false                               | 设置大小写是否敏感         |
| HYPHEN_INSENSITIVE | false                               | 设置连字符不敏感           |
